# Metasploitable2 Penetration Test

## Objective
To identify and exploit vulnerabilities in a vulnerable machine (Metasploitable2).

## Tools Used
- Nmap
- Metasploit
- Kali Linux

## Methodology

### 1. Target Discovery
- Identified target IP address

### 2. Scanning
- Ran Nmap scan to find open ports

### 3. Enumeration
- Analyzed running services

### 4. Exploitation
- Used Metasploit to exploit vulnerabilities

### 5. Post-Exploitation
- Gained shell access

## Results
Successfully gained access to the target system.

## Lessons Learned
- Importance of proper system security
- Risks of vulnerable services
